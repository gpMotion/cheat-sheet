Box Model Reference

CSS Tags

Tag Part of box Description
width:/height: content The content of the box, where text and images appear
padding: Padding Clears an area around the content. The padding is transparent
border: Border A border that goes around the padding and content
margin: Margin Clears an area outside the border. The margin is transparent.
outline-style Outline Outline differs from borders! Unlike border, the outline is drawn outside the element's border, and may overlap other content
